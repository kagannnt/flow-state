# flow-state

> Enter the flow state. 

A Java-based Command Line Interface (CLI) application designed for tracking focus sessions and managing execution history.

---

## Features
* **Focus Session Tracking** – Monitor and log active time intervals directly from the terminal.
* **Local History Persistence** – Stores execution records and session data into a structured JSON database format.
* **Configuration Modes** – Supports custom session rules and customizable intervals.

## Requirements
* Java 17 or higher
* Maven 3.6+

## Quick Start
Build the executable JAR using Maven:
```bash
mvn clean package
